Milestone 1: Variable

1. Var: Declare korar por value change kora jay kintu redecalre kora jay na.
   Example:
   var x = 1;

   if (x === 1) {
   var x = 2;

  console.log(x);
  // Output: 2
  }

  console.log(x);
  // Output: 2

2. Let:
   Declare korar por value change kora jay.
   Redeclare kora jay abar Re-assign o kora jay.

   let x = 1;

   if (x === 1) {
   let x = 2;

   console.log(x);
   // Output: 2
   }

   console.log(x);
   // Output: 1
